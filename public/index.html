PRÉSENTATION

Bébêtopédie est une PWA que j'ai réalisé afin d'aider les joueurs du jeu Animal Crossing: New Horizons à compléter leur bébêtopédie en jeu. Il répertorie donc toute la faune capturable en temps réel dans le jeu et indique aux joueurs où ils peuvent la trouver.

TECHNOLOGIES

    MaterializeCss ReactJS

BlindGen arrow_circle_up

  • Présentation
  • Technologies

PRÉSENTATION

BlindGen est un logiciel console permettant de générer des vidéos de blind test puis, si l'utilisateur le souhaite, l'uploader sur sa chaine youtube. Il permet de choisir le nombre d'extraits du blind test et aussi les catégorie (Film, série, dessin animé, etc…)

TECHNOLOGIES

    ffmpeg youtube-dl .NET Standard 2.0
